**核心修复成功。** 世界不再是「左上角一坨 / 糊成一片」——现在**正立、人尺度、相机站在世界表面上**，画作也回来了、沿墙、眼高、朝内、带框。这是一次实打实的大跨越。

  - 地平线之外一片全黑，空间没有尽头感/边界感，站在里面更像站在一块黄色圆点毯子上，而非 Kusama 式无限空间。

## 3. mesh (A) vs splat (B) —— 建议默认用 **mesh (A)**

| 维度 | A · mesh | B · splat |
| --- | --- | --- |
| 清晰度 | ✅ 锐利、圆点边缘干净 | ❌ 明显糊/涂抹，圆点晕开成一片，还有橙色放射状拉丝（splat 噪点） |
| 房间感 | 都缺墙/顶（并列） | 都缺墙/顶（并列） |
| 稳定度 | ✅ 稳 | 表面有闪烁/拖尾高斯噪点 |
| 画作可读性 | ✅ 好 | ✅ 好（画作是独立平面，两版一样清楚） |
| 像 Marble 网页 viewer | 更接近「成品感」 | 反而更糊，不如 mesh |

**推荐：默认用 mesh (A)。** 它更锐利、更稳，成品感更强；splat 版的糊面和放射噪点会让评委觉得「没渲染好」。

## 4. 下一步建议（供你排期）

按对 demo 观感的影响排序：

1. **补「房间封闭感」** —— 这是现在最影响 wow 的点。要么把这个 mesh 的完整墙/顶找回来（当前可能只加载了地面/局部网格，或世界本身就是开放地形），要么给它一个深色/镜面天穹 + 侧壁，避免「黄色山丘漂在黑虚空」。Kusama 式无限镜屋的关键就是四面包裹 + 反射。
2. **画作贴合** —— 地面是隆起曲面，画作挂点应改为贴合真实墙面/给每幅一个背板，消除「画框切进地形 / 悬浮在黑虚空」。
3. **默认渲染切到 mesh**（把 DEFAULT render 设为 mesh），splat 作为 `?render=splat` A/B 备选保留。
4. 你计划中的 P3（同行者进世界跟随）和多区域连续旅程可继续，但建议先把「房间封闭感」修了，否则每个区域都会是「黑虚空里一片曲面」。
